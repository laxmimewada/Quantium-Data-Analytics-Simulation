# Quantium-Data-Analytics-Simulation
Customer segmentation and uplift testing analysis using Python to evaluate retail store performance.
# Quantium Data Analytics Job Simulation (Forage)

## 📌 Overview
This project is part of a virtual job simulation by Quantium on the Forage platform.

The objective was to analyze customer purchasing behaviour and evaluate the effectiveness of a store trial.

---

## 🎯 Objectives
- Understand customer segments and buying behaviour
- Identify key drivers of sales
- Evaluate store trial performance using control stores
- Provide data-driven business recommendations

---

## 🧰 Tools & Technologies
- Python (Pandas, Matplotlib)
- Google Colab / Jupyter Notebook

---

## 📊 Key Analysis

### 1. Customer Behaviour Analysis
- Segmentation based on life stage and affluence
- Identified key revenue-driving segments

### 2. Trial Store Analysis
- Compared trial vs control stores
- Measured uplift in sales, customers, and transactions

---

## 💡 Key Insights
- Families and premium customers drive most sales
- Customer traffic is the main driver of growth
- Store 77 showed strong uplift, Store 88 showed no impact
 
---

## 📈 Recommendations
- Target family segments with bulk offers
- Expand premium product range
- Roll out strategy to high-performing stores
- Improve strategy for underperforming stores
